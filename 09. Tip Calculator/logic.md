# 💰 Tip Calculator – Super Simple JS Logic

---

## 🥇 STEP 1: Sab cheezein pakdo

// ek variable banao → bill amount input ke liye
// ek variable banao → service (dropdown) ke liye
// ek variable banao → persons input ke liye
// ek variable banao → button ke liye
// ek variable banao → result (total) ke liye

👉 matlab:
“kaun kaun si cheez use hogi → sabko pakdo”

---

## 🥈 STEP 2: Button pe click lagao

// jab user "Calculate" dabaye
// tab hum kaam start karenge 

👉 socho:
“button dabaya → magic shuru”

---

## 🥉 STEP 3: Values lo

// bill amount lo
// tip % lo
// persons lo

👉 example:
100 (bill)
0.10 (tip)
2 (persons)

---

## 🧠 STEP 4: Check karo (IMPORTANT)

👉 agar bill empty hai:
“Enter amount” dikhao

👉 agar tip select nahi kiya:
“Select service”

👉 agar persons empty hai:
default 1 maan lo

---

## 🧠 STEP 5: Tip calculate karo

👉 formula:

```text id="jhx8z4"
tip = bill * service
```

👉 example:
100 × 0.10 = 10

---

## 🧠 STEP 6: Per person calculate karo

👉 agar 2 log hain:

```text id="1qf9hk"
total per person = tip / persons
```

👉 example:
10 / 2 = 5

---

## 🧠 STEP 7: Result dikhao

// screen pe show karo:

👉 Tip Amount:
5

👉 "each" dikhao agar persons > 1

---

## 🧠 STEP 8: Final touch

👉 number ko round karo (optional)

👉 display ko visible karo

---

## 🔁 FINAL FLOW:

// button click
// → values lo

// → calculate karo
// → result dikhao
